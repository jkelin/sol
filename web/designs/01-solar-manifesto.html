01

Compile the framework away

Components become static HTML templates plus precise DOM operations—not a broad rerender loop.

02

Write JavaScript like JavaScript

Writable declarations become signals. Derived constants become computed values. Read and assign without .value.

03

Patch only the lit pixels

Fine-grained dependencies update only the DOM that changed. Setup runs once for each mounted instance.

Navigate

Routes know their shape

Compile-time discovery, typed handles, schemas, path parameters, browser history, and pending states.

Validate

One controller for forms

$form() owns values, issues, and submission—with callable parsers, Valibot, or Zod.

Move

Transitions respect the DOM

Enter and leave classes work on conditionals, keyed lists, and routes—with reduced-motion fallbacks.

03 / Authoring model

Zero
ceremony.
Full
power.

A counter is just declarations, expressions, and assignments. The compiler makes it reactive.

Counter.tsx 
import { $component, mount } from "solix";

const Counter = $component(function Counter() {
  let count = 0;
  const doubled = count * 2;

  return (
    <button onClick={() => count++}>
      {count} / {doubled}
    </button>
  );
});

mount(Counter, document.querySelector("#app")!);
