03 / First light

Normal reads.
Direct updates.

Direct component-body data variables are reactive automatically. Build the interaction with the JavaScript you already know.

For state created outside compiled components, use $signal() and $computed() with their .value APIs.
Counter.tsx / compiled 
import { $component, mount } from "solix";

const Counter = $component(function Counter() {
  let count = 0;
  const doubled = count * 2;

  return (
    <button onClick={() => count++}>
      Count: {count} / Double: {doubled}
    </button>
  );
});

mount(Counter, document.getElementById("app")!);
